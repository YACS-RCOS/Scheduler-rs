### Contributing
- Download rust
- Make a fork
- Clone repository with `git clone https://github.com/YOUR_USERNAME/Scheduler-rs.git`
- Make edits
- Open pull request

### File Structure
- `lib.rs` - External API
- `solver.rs` - Does math to solve an arbitrary set of schedules
- `model.rs` - Internal API's to objects in library
